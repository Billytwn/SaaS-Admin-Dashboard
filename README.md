📌 Présentation
SaaS Admin Dashboard est un tableau de bord d'administration moderne et responsive, conçu pour centraliser la gestion d'une application SaaS. Il offre une vue complète des données métier grâce à des graphiques interactifs, des indicateurs clés (KPIs) et une gestion intuitive des utilisateurs.

✨ Fonctionnalités

👥 Gestion des utilisateurs — ajout, modification, suppression, rôles
📈 Analytics en temps réel — suivi des métriques et indicateurs clés
📊 Graphiques interactifs — courbes, barres, camemberts dynamiques
🔔 Notifications — alertes et événements en temps réel
🌙 Mode sombre / clair — thème personnalisable
📱 Responsive — interface optimisée pour tous les écrans
🔐 Authentification — accès sécurisé par rôle (admin, manager, viewer)
📤 Export de données — téléchargement des rapports (CSV, PDF)

🚀 Installation
Prérequis

Node.js >= 18
npm ou yarn

Cloner le projet
git clone https://github.com/Billytwn/saas-admin-dashboard.git
cd saas-admin-dashboard

Installer les dépendances
npm install

Lancer en développement
npm run dev

Build de production
npm run build
